# Fibonacci Calculator - Лабораторная работа №3

## Описание
Программа для вычисления n-го числа Фибоначчи (0-50) с проверкой входных данных.
Сборка и развертывание в Docker контейнере с использованием GitHub Actions CI/CD.

## CI/CD Pipeline (4 стадии)

### 1. Build
- Проверка кода
- Установка зависимостей
- Компиляция программы

### 2. Test
- Запуск unit-тестов
- Проверка корректности вычислений
- Проверка обработки ошибок
- Тест производительности

### 3. Package
- Сборка Docker образа
- Тестирование образа
- Сохранение образа как артефакта

### 4. Deploy
- Публикация образа в GitHub Container Registry (GHCR)
- Загрузка образа в артефакты

## Локальный запуск

```bash
# Сборка
make

# Тесты
make test

# Docker сборка
docker build -t fibonacci-app .

# Docker запуск
docker run -it --rm fibonacci-app
